## 🚀 Featured Project

### [Community Help](https://github.com/Chuckytin/community-help-pro) 🚧 In development

A local solidarity platform connecting people who want to donate goods (food, clothing, medicine, furniture...) with those who need them, and letting users request help from nearby volunteers.

**Backend highlights:**
- 🤖 **Automatic matching engine** — scores volunteers by real travel time (OpenRouteService), skills, rating and workload. Progressively expands the search radius if no one responds
- 📍 **Geospatial search** with PostGIS — proximity filtering by configurable radius in meters
- 🔐 **JWT + OAuth2** — stateless auth with Google login support
- 💬 **Real-time chat** via WebSocket/STOMP between requester and volunteer
- 📧 **Email digest** — grouped proposal notifications via Brevo SMTP with Thymeleaf templates
- 🗄️ **Flyway migrations** — versioned schema management
- 🐳 **Dockerized** — full stack deployable with Docker Compose

**Frontend highlights:**
- 🗺️ **Interactive map** with React Leaflet showing nearby requests and donations
- ⚛️ **React 18 + TypeScript** — built from scratch, full auth flow (register, OTP verify, reset password, Google OAuth2 callback)
- 🔄 **Real-time chat UI** with STOMP WebSocket client and typing indicators

**Stack:** `Spring Boot 4` · `Java 21` · `PostgreSQL + PostGIS` · `Flyway` · `Docker` · `JWT` · `OAuth2` · `WebSocket` · `React 18` · `TypeScript` · `MapStruct` · `Lombok`
